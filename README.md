# you-quiz in Mossland Metaverse

## 개요
- 예능에 나오는 여러 퀴즈 장르를 한군데에 모아둔 서비스.
- 유저는 퀴즈에 참여하여 자신의 점수를 올리고, 주간, 월간 점수 누적을 바탕으로 MMOC 혹은 MOC를 에어드롭 받는 퀴즈 플랫폼

## 데모
- https://you-quiz.com
- 시간상의 한계로 인물퀴즈만 구현하였습니다.

## 프로젝트 구조
### Client
- Vue 2

### Server
- Node, Express, Typescript

### Proxy
- Nginx Reverse proxy

### DB
- MySQL
